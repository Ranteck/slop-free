# __PROJECT_NAME__

Ultra-strict TypeScript project generated by `create-zero-ts`.

## Quality gates

```bash
npm run check
npm run quality
# Non-destructive aliases for existing projects
npm run zero:check
npm run zero:quality
```

## Rules of thumb

- External input must be `unknown` and parsed with `zod`.
- Read environment variables only through `src/env.ts` (Node/server-side default).
- Do not use `any`, non-null assertions (`!`), or chained assertions (`as unknown as T`).

## Notes

- This template is Node-first (`process.env` in `src/env.ts`). For browser apps, adapt env access to your bundler (for example `import.meta.env`).
- `.github/workflows/quality.yml` is optional. Remove it if you are not using GitHub Actions.
